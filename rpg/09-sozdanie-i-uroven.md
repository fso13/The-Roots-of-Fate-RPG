# Создание персонажа и рост уровня

*Требует ядро. Рекомендуется: [03-derevya-talantov.md](03-derevya-talantov.md), фэнтези-модули.*

## 1. Создание персонажа

### Шаг 1. Концепт

Имя, происхождение, цель — свободно.

### Шаг 2. Атрибуты

Распределите **16 очков** между **Тело, Ловкость, Разум, Воля** (минимум **1**, максимум **5** в каждом на старте).

*Вариант «классика»:* 4/4/4/4 или 5/4/4/3 и т.п.

### Шаг 3. Навыки

- Выберите **один навык** с **рангом 2**, **два навыка** с **рангом 1**, остальные **0** (из списка ядра или расширенного — [fantasy/02-talanty-i-navyki.md](fantasy/02-talanty-i-navyki.md)).
- Если включён **магия**: добавьте **«Магия»** ранг **1** *или* возьмите узел «Первое соглашение» (глава 4) — по правилам кампании.

### Шаг 4. Таланты / дерево

Если включён модуль деревьев: **один стартовый узел** ранга **I** по выбранному архетипу **бесплатно** (или за **3 ОО** с нуля — жёсткий вариант).

### Шаг 5. Ресурсы

- **Стресс макс.** = **6 + Тело** (или по модулю ран).
- **Раны макс.** = **6 + Тело** (или модуль [07-rany.md](07-rany.md)).
- **Искры** (если магия): **3 + Воля**.

### Шаг 6. Снаряжение

Старт: **одно** оружие по деньгам кампании, **доспех** на выбор в рамках допуска ведущего, **набор путешественника** (верёвка, фляга, огниво) + **3 мелких предмета**. Подробнее — глава 5 и [fantasy/03-snaryazhenie.md](fantasy/03-snaryazhenie.md).

## 2. Уровень и «левел-ап»

**Уровень** персонажа — от **1** вверх. После **значимой вехи** (конец арки, 2–4 сессии — решает ведущий) проводится **повышение уровня**.

### Что даёт +1 уровень

| Получаете | Количество |
|-----------|------------|
| **Очки опыта (ОО)** | **3 ОО** за уровень (фикс) *или* **2 + бонус** за ролевую цель — по договорённости |
| **Атрибут** | **один раз на всю кампанию на 4-м и 8-м уровне** (пример): +1 к одному атрибуту (не выше 5 без особых правил) |
| **Навык** | **+1 ранг** к одному навыку (не выше максимума 4; узлы дерева могут поднять потолок) |
| **Талант** | тратите **ОО** на узлы дерева — см. главу 3 |
| **Магия** | при повышении уровня можно вложить ОО в **узел школы**; искры **+0** сами по себе, но узлы могут увеличить максимум искр (если так решено в гл. 4) |

*Упрощение:* на каждом уровне: **3 ОО + один бесплатный ранг навыка** (без покупки ОО за него). Атрибуты реже.

### Таблица (пример прогрессии)

| Уровень | ОО за веху | Заметка |
|---------|------------|---------|
| 1 | — | создание |
| 2 | 3 ОО | первый ап |
| 3 | 3 ОО | |
| 4 | 3 ОО + **+1 атрибут** | по опции выше |
| … | … | |

Ведущий может заменить **уровни** на **только ОО** без цифры уровня — тогда этот файл читать как «что даёт пакет ОО».

## Пример из партии: «Стеклянные волки»

Группа закрыла главу о канале. Ведущий объявляет **уровень 2**.

**Кай** получает **3 ОО**, тратит **2** на узел **«Тихий шаг II»**, **1** копит. Бесплатно кладёт **+1 ранг** в **«Скрытность»** (1→2).

**Мира** кладёт бесплатный ранг в **«История»**, **3 ОО** откладывает на магический узел (покупка позже).
