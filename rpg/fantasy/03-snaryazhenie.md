# Фэнтези-модуль: расширенное снаряжение

*Дополняет [05-snaryazhenie.md](../05-snaryazhenie.md). Колонка «Бонус» — слагаемое к **(итог атаки − ЧЦ) + бонус** (глава 2).*

## Оружие ближнего боя

| Название | Бонус | Примечание |
|----------|------|------------|
| Кинжал | +1 | Ловкость |
| Короткий меч | +2 | Тело или Ловкость |
| Длинный меч | +2 | Тело или Ловкость |
| Двуручный меч | +3 | Тело |
| Топор боевой | +2 | Тело |
| Булава | +2 | Тело, пробивает лёгкий доспех — нарратив |
| Копьё | +2 | с **Рядом** бьёт как дальняя **+1** (метательно) — по договорённости |
| Посох | +1 | можно вложить кристалл: **+0**, но даёт **+1** к **Магия (ближняя)** при атаке заклинанием |
| Кнут | +1 | Ловкость, **Рядом** |
| Когти | +1 | без рук, монстры |

## Дальнобойное

| Название | Бонус | Дистанция |
|----------|------|-----------|
| Лук лёгкий | +2 | Даль |
| Лук тяжёлый | +3 | Даль |
| Арбалет | +3 | Даль, перезарядка — 1 ОД |
| Метательный нож | +1 | Даль, расходуемый слот |

## Доспехи

| Тип | К ЧЦ защиты | Заметка |
|-----|-------------|---------|
| Стёганный | +0 | под одеждой |
| Кожаный | +1 | |
| Шкура | +1 | дикари |
| Кольчуга | +2 | |
| Чешуя | +2 | |
| Латы | +3 | |
| Щит малый | +1 | к щиту |
| Щит большой | +2 | нужны свободные руки |

## Инструменты и прочее

| Предмет | Эффект |
|---------|--------|
| **Зелье лечения** | вне боя или **1 ОД**: снять **1 рана** (ограничить по количеству) |
| **Антидот** | автоуспех против одного известного яда |
| **Факел** | свет, **+0** как импровизированное оружие |
| **Серебряное оружие** | нарратив против **проклятых/нежити** — **+1** к бонусу раз за сцену |
| **Волшебный фокус** | **+1** к **Магия (дальняя)** **или** **(ближняя)** на выбор при экипировке |

## Качество

**Обычное / мастерское / артефакт:** **+0 / +1 / +2** к **бонусу** в формуле урона или особое свойство — не стакается без согласия ведущего.

---

## Пример из партии: «Стеклянные волки»

**Гарр** берёт **копьё** и **щит большой**: ЧЦ защиты высокий, но **2 ОД** на «войти в Рукопашную из Даль» стоит дорого — игрок планирует позицию заранее.
