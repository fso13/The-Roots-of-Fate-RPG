# Модуль: получение и учёт ран

*Опционально. Требует ядро; для боя — [02-boy.md](02-boy.md). Расширение: **ранения частей тела** — по желанию стола.*

Число **нанесённых ран** за атаку = **урон** по формуле главы 2: **(итог атаки − ЧЦ защиты) + бонус оружия** (и модификаторы критов, если включены).

Если модуль **выключен**, при попадании просто отнимайте **раны** как единый ресурс (как в главе 0) без дополнительных правил.

## 1. Полоска ран

**Максимум ран** = **6 + Тело** (или **4 + Тело** для «мягкой» кампании). При **0 ран** персонаж **выбит**: без сознания или смерть — по договорённости стола.

## 2. Типы ран (по желанию)

Когда атака наносит урон, ведущий или атакующий помечает **тип** (для ролевого эффекта):

| Тип | Пример последствия (нарратив, не стакается без модуля критов) |
|-----|----------------------------------------------------------------|
| **Лёгкая** | царапина, −0 к броскам |
| **Тяжёлая** | шок: до конца сцены **−1** к первому броску Тела |
| **Критическая** | только если сработал [08-krity-i-promahi.md](08-krity-i-promahi.md) или особое оружие |

Игроки могут **игнорировать** типы и вести только число — модуль всё равно полезен для «сколько до падения».

## 3. Стабилизация (общий урон)

Вне боя или **1 ОД** в бою: проверка **2d6 + Разум + «Медицина»** (или Тело + Выживание) **≥ 10** — **снять 1 рану** у союзника **Рукопашная/Рядом** (не у себя без талантов).

---

## 4. Ранения частей тела *(опционально)*

Используйте, если хотите **не только** общую полоску, но и **руки, ноги, голову**.

### 4.1. Зоны удара

| **1d6** | Часть тела |
|---------|------------|
| **1** | **Голова** |
| **2** | **Грудь / спина** (торс) |
| **3** | **Рука с оружием** (ведущая) |
| **4** | **Другая рука** |
| **5** | **Нога** (ведущая / опорная) |
| **6** | **Другая нога** |

*Существа без рук/с четырьмя лапами:* ведущий переназначает строки **3–6** (передние/задние лапы, хвост и т.д.).

### 4.2. Когда определять зону

| Способ | Как |
|--------|-----|
| **Случайно** | После **любого успешного попадания**, если стол включил локализацию: бросьте **1d6** по таблице выше. |
| **Прицельный удар** | **До** броска атаки объявите зону. **−2** к **итогу атаки** (голова: **−3**). Если попадание есть — урон считается как обычно, но **последствие** (п. 4.3) применяется **к этой** зоне и на **одну ступень суровее** (см. ниже). |
| **Крит по модулю [08-krity-i-promahi.md](08-krity-i-promahi.md)** | Можно бросить зону и дать **дополнительный** эффект по зоне без прицела. |
| **Удобное правило** | Локализацию бросаете **только** если урон **≥ 3** за удар **или** был прицел **или** крит — чтобы не кидать каждый раз. |

### 4.3. Состояния конечности / зоны

Одна и та же зона может быть **целой**, **повреждена**, **выведена**. Отметьте на листе.

| Состояние | Что значит |
|-----------|------------|
| **Целая** | нет штрафов |
| **Повреждена** | **−1** к проверкам, где участвует эта часть (рука — атаки/ловкость рук; нога — движение, Уклонение; голова — Разум/Воля на коротком отрезке; грудь — Тело) |
| **Выведена** | часть **недоступна**: рука роняет оружие / не держит щит; нога — **половина ОД** на **Сдвиг** или нельзя **бежать**; голова — **1 стресс** за раунд до стабилизации |

**Голова — особый случай:** второе попадание в **повреждённую** голову (или одно тяжёлое с прицелом) — ведущий может объявить **нокаут** или **смерть** (по жёсткости кампании).

**Торс:** «повреждён» = **−1** к Телу на проверках до лечения; «выведен» = персонаж **выбывает** из боя (кровотечение, шок).

### 4.4. Связь с числом ран

- **Простой вариант:** локализация — только **ролевой** штраф; **раны** снимаются с общего запаса как раньше.
- **Жёсткий вариант:** при переходе зоны в **«выведена»** дополнительно **−1** к общей полоске ран (один раз за эпизод с этой зоны).

Выберите **один** вариант на кампанию.

### 4.5. Лечение локальных ран

| Действие | Проверка / условие | Эффект |
|----------|-------------------|--------|
| **Перевязка / шина** | **1 ОД** вне боя или **2 ОД** в бою; **2d6 + Медицина ≥ 10** | **повреждена** → **целая** **или** **выведена** → **повреждена** (на выбор, одна зона) |
| **Покой и уход** | между сессиями / **длинный отдых** | все **повреждённые** зоны → **целые**, если не было **выведено** |
| **Хирургия** | **Медицина ≥ 13**, инструменты, **1 час** | **выведена** конечность без ампутации → **повреждена** |
| **Магия** | заклинания **Сотворения** (глава 4, списки) | как **сильное лечение**, часто снимает и зону, и **1–2 раны** общих |
| **Зелье лечения** | по снаряжению | обычно **+1 рана** общая; по согласованию — ещё **одна** зона на ступень вверх |

Ампутация и протезы — **нарратив** и домашние правила.

## 5. Без лишнего счёта

Не вводите одновременно **5 дебаффов**. Если нужен штраф — один на сцену на персонажа **или** одна зона в фокусе.

---

## Пример из партии: «Стеклянные волки»

У **Миры** **2 раны** из **9**. **Гарр** стабилизирует: **Медицина**, снимает **1 рану**.

В другой сцене **Кай** получает прицельный удар в **ногу** (**−2** к итогу атаки врага, но попал). Зона **нога** → **повреждена**; Кай **−1** к первому **Сдвигу** в сцене. Между боями **перевязка** снимает пометку.
